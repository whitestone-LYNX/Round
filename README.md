# Round

Here's Round — a single-file, dependency-free HTML app. It's a real-time delay looper, not a record-and-playback-later tool, which turns out to be the simpler and more musical approach: your live playing runs continuously through one or more independent delay lines built on the Web Audio API, so each "voice" is just you, offset by however many seconds you set.

This project took time to make, please consider showing it to your friends and even family! If you find any bugs/issues you can just talk to me!
